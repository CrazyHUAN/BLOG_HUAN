# window下配置 Bash 默认打开路径

  1. 右键->属性->快捷方式
  2. 目标: 删除 --cd-to-home
  3. 起始位置: 修改为想要开启的默认路径

原始配置
目标:       "C:\Program Files\Git\git-bash.exe" --cd-to-home
起始位置:    %HOMEDRIVE%%HOMEPATH%
